## Acceptance criteria

- **Vaihe 3**: Trade offers feel fair; reactions enrich drama → **PARTIAL PASS**

## What landed in this iteration

### Engine layer (Vaihe 3)
- `TradeEvaluator`: Jimmy Johnson + need bonus + 65% future-pick discount + 4 GM-personalities (`evaluate`, `valueOf`, `aiInitiatedOfferConsidered`)
- `DraftReputation` `@Model`: per-season ownerTrust / fanMood / lockerRoomMood + `MediaNarrative` enum
- `ReactionsEngine`: 4-actor reactions with **selectivity matrix** (silent on B Solid, 2 actors on A/C, all 4 on A+/D), QB / locker-room / rival bonus triggers
- `RoundRecapBuilder`: cumulative summary per round with picks + reputation deltas + league steals
- `DraftDramaEngine`: stealOfTheDraft / roundTransition / gemMoment / userPickIncoming / finalPick

### UI layer (Vaihe 4)
- `ReactionToast`: bottom-edge cascade, sentiment-tinted, FIFO 2 s dwell — **VERIFIED in simulator**
- `DramaOverlayView`: ESPN-style steal banner / round-transition fade / gem flash / picks-incoming pulse — **STEAL BANNER VERIFIED in simulator**
- `RoundRecapSheet`: large modal with picks list, reputation deltas, league steals
- `TradeOfferBanner`: standalone scaffold (not yet wired to coordinator's pendingTradeOffer)
- `DraftDayView`: ZStack with drama overlay above 3-pane content + reaction toasts at bottom + BgDraft stadium background opacity 0.15

## Issues found

| # | Issue | Severity | Plan |
|---|-------|----------|------|
| 1 | Round Recap modal does not appear when round transitions during a skip | high | `autoAdvanceUntil` mutates `currentPickIndex` directly instead of calling `advance()`, so `triggerRoundRecap` is never fired during skips. Move advance() call into the skip loop. |
| 2 | Pick Grade C ("Reach") displayed alongside "STEAL +43" can confuse the user | medium | Logic is correct (Reach because Need ≤ 0.3 even though value Δ is positive) but the dual signaling is unclear. Either drop the Steal label when grade is Reach, or rebrand to "VALUE +43" |
| 3 | Trade offer flow not yet user-visible (engine is ready, UI banner stubbed) | medium | Wire `coordinator.pendingTradeOffer` + integrate `TradeOfferBanner` into DraftDayView ZStack |
| 4 | Reactions only fire for user team picks; AI team picks generate no reactions | low | Design says AI-team picks can trigger media reactions if rivalDivisionStarter — implement |
| 5 | RoundRecap "before/after reputation" deltas always 0 because `beforeReputation` is nil | low | Snapshot reputation at round start so deltas are meaningful |
| 6 | Final pick "🎉 Mr. Irrelevant" overlay not yet observed in playtest | low | Run draft to completion to verify |
