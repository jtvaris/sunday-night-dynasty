## Vaihe 2 acceptance criterion

> Smart pick achievable from UI alone; reasons visible.

**Verdict: PASS** — Pick Sheet shows Pick Grade (A+/A/B/C/D), NEED tag, scout-confidence stars, BB rank, Steal/Reach delta. War Room shows draft capital breakdown + dynamic scout chatter.

## What landed

| Item | Description |
|------|-------------|
| `DraftIntel` engine | Public board ranks (1..N from `trueOverall × positional value`), scout-confidence (1-5 stars from report depth + combine/interview/proday), reach indicator, team need scores |
| Persistence fix (V2.0) | `WeekAdvancer.persistDraftClass(_:to:)` flushes prospects to SwiftData on every generate path; `ScoutingHubView.loadData()` now: SwiftData → re-gen + persist; `DraftDayCoordinator.loadData()` priority: SwiftData (truth) → in-memory (orphan, persist) → generate + persist |
| Big Board | Ranks now contiguous 1..N from `coordinator.publicBoardRanks`; UUID dedupe on the available pool; scout-confidence stars visible; gold-tinted background for high team-need positions |
| Pick Sheet | Per-row Pick Grade chip (A+/A/B/C/D); NEED tag for high-need positions; STEAL/REACH/FAIR label; scout stars; gold border on gem candidates |
| Sticky header | Round-relative "Pick 1/32 · Overall 1"; NEEDS strip with 5 top positions tinted by intensity; gold warning pulse when user pick ≤3 away |
| War Room | "Your Draft Capital" lists every remaining user pick (Rd, # number, points) with prominent total; Scout Chatter adapts to mode (on-clock prompt, steal/reach reactions to last pick, "get ready" warning) |

## Issues identified during the run

| # | Issue | Resolution |
|---|-------|-----------|
| 1 | Initial Big Board / Combine empty after restart even though pelin status näytti "Combine Reviewed" | Root cause: WeekAdvancer.currentDraftClass static lost on restart; SwiftData was never loaded back. Fixed via `persistDraftClass`. |
| 2 | BB rank uniform "#2" for all rows | Was using `prospect.draftProjection` (1-7 round) instead of `DraftIntel.publicBoardRanks` (1..N pick number). Fixed. |
| 3 | BB rank hyppii (1, 4, 5, 7...) | Old persisted prospects from earlier sessions occupied ranks. Fixed by computing ranks only over the visible available pool (declared + not-drafted + UUID-deduped). |
| 4 | Pick capital "468 pts" feels small | Confirmed correct — Packers' 6 picks are concentrated in late rounds (Rd 2 #63 → Rd 7 #223). User now sees the full breakdown so the number is contextualised. |
