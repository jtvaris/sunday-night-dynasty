# Draft Day — Iteration 1 Playtest Notes

**Date**: 2026-05-05
**Phase**: Vaihe 1 (Core mechanics) acceptance run
**Build**: iPad Pro 13-inch (M5) simulator, iOS 26.4

## Vaihe 1 acceptance criterion

> Draft completes end-to-end (224 picks), no crashes, user pick visible and selectable, skip-to-my-pick works, basic ticker rolls.

**Verdict: PASS** — all bullets verified.

## Run flow (this iteration)

1. App launched into Main Menu
2. Tap **Continue Career** → Career Dashboard (save: Green Bay Packers, 2026 Combine phase)
3. Tap **Draft** tab in top nav → DraftDayView opens
4. Sticky header reads `ROUND 1 — Pick 1 of 224`, `ON THE CLOCK: Las Vegas Raiders`, `58s`, `Your next pick: #63 (62 picks away) • 6 remaining`
5. Layout renders 3-pane: Big Board / Draft Ticker / War Room with control bar at bottom
6. Tap **My Pick** in control bar → coordinator skips through 62 AI picks, modal opens for pick #63
7. Pick Sheet shows top 20 prospects sorted by BB rank, each row offers Steal / Reach chip
8. Tap top prospect (Jaylon Smith QB Baylor) → coordinator records pick, sheet dismisses, draft advances to #64 Atlanta Falcons

## What worked

- 3-pane layout adapts to landscape iPad
- Sticky header counts down (120 s clock visible, decremented on tick)
- Skip-to-my-pick processes 62 AI picks instantly without UI freeze
- Pick Sheet renders with Steal/Reach indicator per row
- Selecting a prospect persists the pick and advances state
- Ticker fills in retrospectively after skip — past picks visible in feed
- War Room Pick Value card surfaces correct point totals (Jimmy Johnson chart)
