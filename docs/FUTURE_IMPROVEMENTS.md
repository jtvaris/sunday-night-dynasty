# Future Improvements

Ideas and features that are out of scope for current sprint but should be considered for future development.

## Coordinator Schemes — Realistic Expansion

Current: 5+5 simplified scheme system. Future: expand to realistic NFL scheme depth:

### Offensive Schemes (expanded)
- West Coast (short passing tree, timing routes)
- Erhardt-Perkins (Patriots-style, flexible route concepts)
- Air Coryell (vertical passing, deep shots)
- Shanahan Wide Zone (outside zone run, bootlegs, play-action heavy)
- Spread (shotgun-based, tempo, RPO)
- Air Raid (4-5 WR sets, quick reads)
- Power Run / Smashmouth (gap scheme, pulling guards, I-formation)
- Pistol/Option (read option, triple option concepts)
- Run & Shoot (4 WR, hot routes, audibles)
- Custom hybrid schemes (mix elements from multiple systems)

### Defensive Schemes (expanded)
- 4-3 Over/Under variations
- 3-4 Even/Odd variations
- Fangio/Vic 2-high shell
- Tampa 2 (Cover 2 with athletic MLB)
- Cover 3 / Seattle style (single high, press man)
- Cover 1 Robber (man coverage + robber safety)
- Hybrid / Positionless defense (multiple looks)
- Amoeba / Psycho packages (no down linemen)
- 46 Defense (8 in the box, aggressive)
- Pattern matching zone concepts

### Scheme Details
Each scheme would include:
- Formation tendencies (% shotgun, % under center, % pistol)
- Personnel groupings (11, 12, 21, 22 etc.)
- Run/pass ratio tendencies
- Blitz frequency
- Coverage shell distribution
- Ideal player archetypes per position
- Historical NFL teams that ran this scheme
- Coordinator familiarity rating
